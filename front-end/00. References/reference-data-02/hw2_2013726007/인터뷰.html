인터뷰 결과
1.각각의 방법 장점 단점
(네이버 다음)웹페이지 -> 단점 : 키워드가 좀만 바뀌어도 검색이 어려워진다
: 특정 조건(빨간날, 새벽) 검색이 어렵다
장점 : 말풍선에 나오는 정보들이 한눈에 들어온다
: 지도기반으로 다양한 기능을 쓸 수 있다(내위치 내비)
휴일 약국 지키미 -> 단점 : 검색 조건이 너무 많다
: 지도도 안나와서 위치를 알기 힘들다
: 사이트 오류가 너무 많다 (검색 결과가 안나오거나 다른 지
역이 나오는)
: 인터페이스가 눈에 확 들어오지 않는다
장점 : 검색결과 리스트가 눈에 확 들어온다
약국찾기 대화형 프로토타입 ->
단점 : 데이터의 양이 많지 않다
: 현재 위치가 뜨질 않아 위치 가늠이 어려움
장점 : 검색조건이 간단하고 사용하기 쉬운 인터페이스라 검색하기
가 쉬웠음
: 요일을 정하고 시간을 정하는 부분이 있어 주말에 하는지
새벽에 하는지 검색 조건을 달아주기가 매우 편함
: 검색을 빠르게 할 수 있어 좋았다.
2. 추가해줬으면 하는기능?(많이 나온 top 5)
1.지도에 내위치 기능이 추가 되었으면 좋겠다.
2.시,도를 입력하는 부분 샐렉트 박스로 바꿨으면 좋겠다.
3.검색 성공,실패 결과 피드백이 확실했으면 좋겠다, 또한 검색중일 때 로딩 표시가 있었음 좋
겠다.
4.검색의 양이 많았음 좋겠다
5.왼쪽 텍스트 박스에 자세한 정보창을 넣지 않고 나온 정보들의 리스트를 만들어 그 리스트
를 누르면 해당 마커로 이동하는 기능이 있었으면 좋겠다.
